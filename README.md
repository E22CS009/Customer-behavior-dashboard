# 📊 Customer Behavior & Spending Analysis Dashboard

## 📌 Project Overview

This project analyzes customer data to understand spending patterns, segmentation, and satisfaction levels using Google Sheets.

## 🛠 Tools Used

* Google Sheets
* Pivot Tables
* Charts & Dashboard

## 📈 Key Analysis

* Revenue by Membership Type
* Customer Segmentation (High, Medium, Low Value)
* Recency vs Spending Trends
* Impact of Discounts on Spending
* Customer Distribution by City

## 💡 Key Insights

* High-value customers contribute the most revenue
* Recent customers spend more than inactive customers
* Discounts influence customer spending behavior
* Customer base is concentrated in major cities

## 🚀 Conclusion

This dashboard provides actionable insights into customer behavior and helps in decision-making for marketing and retention strategies.
